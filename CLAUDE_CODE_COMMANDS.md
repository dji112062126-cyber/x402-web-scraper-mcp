# Claude Code 斜杠命令大全

## 对话与帮助

| 命令 | 功能说明 |
|------|---------|
| `/help` | 打开帮助面板，显示所有可用命令和快捷键 |
| `/clear` | 清除当前对话历史，开始全新对话 |
| `/reset` | 完全重置会话（包括工作树、任务等） |
| `/compact` | 压缩对话上下文，自动总结早期内容以释放 token 空间 |
| `/context` | 显示当前上下文窗口使用情况 |
| `/resume` | 恢复之前被压缩或中断的对话 |

## 文件与代码操作

| 命令 | 功能说明 |
|------|---------|
| `/add-dir` | 添加目录到工作区，让 Claude 可以访问指定目录下的文件 |
| `/init` | 为当前项目初始化 CLAUDE.md 文件，自动分析代码库并生成文档 |

## 模式切换

| 命令 | 功能说明 |
|------|---------|
| `/plan` | 进入计划模式 — Claude 先设计实施方案，获得批准后再写代码。适合复杂功能开发 |
| `/thinking` | 切换思考模式 — 显示 Claude 的推理过程，让决策更透明 |
| `/fast` | 切换快速模式 — 使用 Opus 模型但输出更快，适合快速问答 |
| `/ultra` | 切换 UltraCode 模式 — 多智能体全面审查，用工作流完成所有实质性任务 |

## 自动化与任务

| 命令 | 功能说明 |
|------|---------|
| `/loop` | 定时循环执行某个命令，例如 `/loop 5m /code-review` 每 5 分钟运行一次代码审查 |
| `/tasks` | 查看当前任务列表和进度 |
| `/workflows` | 查看正在运行的多智能体工作流状态 |
| `/cron` | 管理定时任务（创建、列出、删除持久化的定时任务） |
| `/output` | 查看后台任务的输出 |

## 版本控制 (Git)

| 命令 | 功能说明 |
|------|---------|
| `/commit` | 自动生成提交信息并提交当前更改 |
| `/pr` | 创建或管理 Pull Request |
| `/review` | 评审 Pull Request，分析代码变更并给出反馈 |
| `/worktree` | 创建隔离的 Git 工作树，在独立分支中实验而不影响主分支 |

## 代码质量

| 命令 | 功能说明 |
|------|---------|
| `/code-review` | 代码审查 — 检查 diff 中的 bug、安全问题和优化机会。支持 `--fix` 自动修复 |
| `/simplify` | 简化代码 — 审查变更中的复用、简化、效率优化，并自动应用修复 |
| `/security-review` | 安全审查 — 对当前分支的变更进行全面的安全检查 |
| `/verify` | 验证改动 — 运行应用并观察行为，确认代码变更是否真正有效 |
| `/lint` | 运行代码检查工具（ESLint、Pylint 等）并修复问题 |

## 测试

| 命令 | 功能说明 |
|------|---------|
| `/test` | 运行测试套件，自动检测测试框架并执行 |
| `/test --watch` | 监听模式运行测试，文件变更时自动重新运行 |

## 调试与问题排查

| 命令 | 功能说明 |
|------|---------|
| `/doctor` | 诊断 Claude Code 安装和配置问题 |
| `/bug` | 提交 bug 报告 |
| `/status` | 显示当前会话状态（模式、工作树、后台任务等） |
| `/terminal-setup` | 配置终端集成（如 iTerm2、tmux） |

## 配置与权限

| 命令 | 功能说明 |
|------|---------|
| `/config` | 打开设置面板，修改主题、模型等配置 |
| `/permissions` | 管理工具调用权限（允许/拒绝特定命令或文件访问） |
| `/update-config` | 配置 settings.json（钩子、权限、环境变量等） |
| `/hooks` | 管理钩子脚本（在特定事件前后自动触发） |
| `/keybindings` | 自定义键盘快捷键和组合键绑定 |
| `/model` | 切换使用的 Claude 模型（Opus / Sonnet / Haiku） |

## 外部集成

| 命令 | 功能说明 |
|------|---------|
| `/mcp` | 管理 MCP (Model Context Protocol) 服务器连接 |
| `/ide` | 管理与 IDE（VS Code / JetBrains）的集成状态 |
| `/login` | 登录/切换 Anthropic 账户 |
| `/logout` | 注销当前账户 |

## 技能 (Skills)

| 命令 | 功能说明 |
|------|---------|
| `/deep-research` | 深度研究 — 多源 Web 搜索、获取资料、对抗性验证、生成引用报告 |
| `/claude-api` | Claude API 开发辅助 — 构建/调试/优化 Anthropic SDK 应用 |
| `/run` | 启动并检查项目应用，确认改动是否有效 |
| `/fewer-permission-prompts` | 扫描对话记录，自动添加常用只读命令的权限允许列表 |

---

> 提示：在 Claude Code 中直接输入 `/` 即可看到所有可用命令的自动补全列表。
