# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please email dji112062126@gmail.com.

## Payment Security

This service uses the x402 protocol for payments. All transactions are verified
on-chain on Base (chain ID 8453) against the USDC contract at
`0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913`.

No private keys are stored on the server. The recipient address is read-only.
